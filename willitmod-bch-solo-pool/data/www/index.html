Bitcoin Cash (BCH) Solo Pool

Important: update data/config/ckpool.conf and replace CHANGEME_BCH_NODE_APP_PASSWORD with your BCH node app password.

Miner connection

  • URL: stratum+tcp://<umbrel-ip>:3333
  • Username: <payout-address>[.<worker>]
  • Password: any value

Status

If ckpool is running, it will usually write status files into this folder (served here).

  • /pool/pool.status
  • /pool/pool.workers

RPC target

Default is umbrel.local:28332 (BCH node RPC) with ZMQ at umbrel.local:28334.
